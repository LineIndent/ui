# Button

Displays a button or a component that looks like a button.

# Installation

Copy the following code into your app directory.

--INSTALL(["Button", "buridan add component button"])--

# Usage

--USAGE(button)--

# Anatomy 
Use the following composition to build a `Button`

--ANATOMY(button)--
