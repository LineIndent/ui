# Breadcrumb

Displays the path to the current resource using a hierarchy of links.

# Installation

Copy the following code into your app directory.

--INSTALL(["breadcrumb", "buridan add component breadcrumb"])--


# Anatomy

--ANATOMY(breadcrumb)--
