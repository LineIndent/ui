# Avatar

Displays a user's profile picture, initials, or fallback icon.

# Installation

Copy the following code into your app directory.

--INSTALL(["Avatar", "buridan add component avatar"])--

# Usage

--USAGE(avatar)--

# Anatomy 
Use the following composition to build a `Avatar`
